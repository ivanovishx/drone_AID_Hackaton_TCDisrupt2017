      class TimelineElement

      Package:dji.sdk.missioncontrol
      Doc Key:DJIMissionControl_DJIMissionControlTimelineElement
      Description:

      It is a protocol that defines the behavior of elements that may be run by the Mission Control in its timeline. When implementing your own elements, it is critical that you implement the Element -> Mission Control communication methods as relevant to your situation.
